# Deskripsi Aplikasi

CineTIX adalah aplikasi pemesanan tiket bioskop berbasis Android yang dibuat menggunakan Java di Android Studio. Aplikasi ini terinspirasi dari aplikasi bioskop modern seperti TIX ID dan CGV, namun menggunakan data lokal/fiktif tanpa database online.

Aplikasi memungkinkan pengguna untuk:
- Melihat daftar film yang sedang tayang
- Melihat detail film
- Memilih jam tayang
- Memilih studio
- Memilih kursi bioskop
- Menambahkan snack
- Melakukan pemesanan tiket
- Melihat tiket digital dengan QR Ticket UI

Selain itu aplikasi juga memiliki fitur modern seperti Bottom Navigation, Fragment, SharedPreferences, RecyclerView, dan Implicit Intent untuk membuka IMDb dan Netflix.
